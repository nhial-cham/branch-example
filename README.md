مهندس برمجيات متخصص في تطوير تطبيقات الويب المتكاملة، بخبرة واسعة في لغتي JavaScript وPython. أمتلك مهارات عالية في بناء واجهات برمجة تطبيقات RESTful وGraphQL قابلة للتوسع، وتصميم بنى برمجية نظيفة، وتقديم برمجيات عالية الجودة وسهلة الصيانة.

لديّ خبرة عملية في تطوير أنظمة الواجهة الخلفية باستخدام Node.js وExpress وDjango وFlask، والعمل مع قواعد البيانات العلائقية وغير العلائقية. أركز على كتابة برمجيات نظيفة وقابلة للاختبار، ملتزمًا بمبادئ SOLID وأفضل الممارسات في هندسة البرمجيات.

شغوف بحل المشكلات الواقعية باستخدام التكنولوجيا، وأستمتع بتصميم أنظمة فعّالة، وتحسين الأداء، والارتقاء بجودة البرمجيات باستمرار. ألتزم بالتعلم المستمر، والتعاون، وتقديم حلول موثوقة تُحدث أثرًا ملموسًا.
